Need help with FastResume?

If you face any issue with the resume builder or have ideas to improve it, you can reach out directly by email.

FastResume is a side project built by a student. Replies may not be instant, but genuine feedback from students is always appreciated.

Can you edit my resume personally? FastResume is mainly a tool for you to generate your own resume. Personal resume editing is not guaranteed, but feedback is welcome.
I found a bug in the builder. What should I do? Please send a short description plus a screenshot (if possible) so the issue can be reproduced and fixed.
Can I suggest new features? Yes! Ideas like new templates, extra sections (achievements, hackathons), or better export options are always welcome.
